man <nome do comando> = exibe a página do manual referente ao comando
whatis <nome do comando> = exibe uma descrição do comando
mkdir <nome> = cria uma pasta
touch <nome>.<extenção> = cria um arquivo
pwd = mostra o diretório atual
ls = mostra as pastas e arquivos no diretório atual
ls -a = mostra as pastas e arquivos no diretório atual, incluindo ocultos
cd .. = navega para o diretório pai
cd <nome> = navega para o diretório
nano <nome do arquivo> = edita o arquivo no terminal
chmod <permissões> <arquivo> = altera as permissões de leitura, escrita e execução para o dono, usuários do grupo e outros usuários
